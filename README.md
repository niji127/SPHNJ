# SPHNJ
SPH 2d solver<br />
### Features:   
- explicit weakly-compressive-SPH<br />
- gas-liquid-solid three-phase-coupling<br />
- non-linear constitutive relationship<br />
- multi-resolution time-stepping<br />
- GPU acceleration on CUDA<br />
